“EFFICIENT INFORMATION MANAGEMENT IN AGRICULTURE USING IOT”

• Efficient information management is critical for modern agriculture, and the Internet of Things (IoT) has revolutionized the way agricultural operations are managed. Data gathering can extend the lifetime of sensor nodes in WSNs by minimizing redundant data transmission. In this case, data combination presents an important method for reducing the amount of redundant data, reducing the number of transmissions, and minimizing the power consumption of the nodes in the network. Data merging can be accomplished along a route starting from sensor nodes to sink nodes. Gathering data in agricultural applications, including humidity, weather,temperature, and soil moisture, can minimize the power consumption of environment sensor nodes by spatially correlating the field and redundant data of the sensor nodes when the temperature does not change significantly, especially during night time.
So, we have come up with a solution to efficiently gather or collect the information with different schemes and techniques implemented using IoT. They are:
Data aggregation: Data aggregation involves combining multiple data points into a single data point. This can help to reduce the amount of data being transmitted to the cloud. For example, instead of sending the soil moisture level of each plant in a field, the average soil moisture level of the field can be sent.
Data filtering: Data filtering involves sending the data only when there is change in the data. This can help to reduce the amount of data being transmitted to the cloud. For example, transferring the data only when there is a significant change in the data.
Data sampling: data sampling involves collecting data at a lower frequency to reduce the amount of data being transmitted to the cloud. For example, instead of collecting data every minute, data can be collected every 5 or 10 minutes.
